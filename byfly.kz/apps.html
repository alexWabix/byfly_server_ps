Как установить приложение ByFly Travel

Для iPhone
  1. Перейдите по ссылке TestFlight.
  2. Установите приложение TestFlight из App Store, если оно не установлено.
  3. Откройте TestFlight и примите приглашение на установку ByFly Travel.
  4. Нажмите «Установить» и дождитесь завершения установки.
Скачать для iPhone
Для Android
  1. Скачайте APK-файл по ссылке APK.
  2. Разрешите установку приложений из неизвестных источников в настройках телефона.
  3. Найдите скачанный файл и нажмите «Установить».
  4. После завершения установки откройте приложение и войдите в аккаунт.
